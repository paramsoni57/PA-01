# Asset Folder Guide

Use these folders when updating images:

- `assets/brand/` - logo and brand images
- `assets/banners/` - homepage and product-page banner images
- `assets/maps/` - map/location images
- `assets/products/categories/` - product category thumbnails, including `digital-motor-starter.jpg`
- `assets/products/digital-starters/` - individual digital starter product images

Keep file names lowercase and use hyphens instead of spaces, for example:

`three-phase-digital-starter-7-5hp.jpeg`
